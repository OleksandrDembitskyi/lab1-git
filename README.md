# Lab1-Git — Лабораторні роботи з управління ІТ-проєктами

## Проєкт
Базовий React-застосунок, створений для виконання серії лабораторних робіт. Слугує основою для вивчення Git, тестування, збірки, CI/CD та моніторингу.

## Технології
- React 19
- Vite 7
- JavaScript (ES6+)
- ESLint / Prettier
- **Тестування:** Vitest, Playwright

---

## 📁 Лабораторна робота №1 — Git та гілки
**Тема:** Управління кодом та організація робочого простору в Git

### Виконано:
- Створено структуру проєкту (`.gitignore`, `README.md`)
- Робота з гілками (`main`, `develop`, `feature/*`)
- Створено та змерджено Pull Request з код-рев'ю
- Вирішення merge-конфлікту

### Команди:
```bash
git branch                 # перегляд гілок
git checkout -b feature/initial-layout
git add .
git commit -m "feat: ..."
git push origin feature/initial-layout
```

---

## 🧪 Лабораторна робота №2 — Тестування
**Тема:** Забезпечення якості програмних обєктів через багаторівневе тестування

### Додано:
- Простий калькулятор (функції `add`, `subtract`, `multiply`, `divide`)
- **16 Unit-тестів** з Vitest (100% покриття бізнес-логіки)
- **5 E2E-тестів** з Playwright (перевірка UI)
- Аналіз покриття коду (Code Coverage)
- Навмисне внесення помилки (демонстрація падіння тесту)

### Команди для тестування:
```bash
npm run test:unit                 # запуск Unit-тестів
npm run test:unit:coverage        # покриття коду
npm run dev                       # запуск сервера
npx playwright test               # E2E-тести
npx playwright show-report        # перегляд звіту
```
